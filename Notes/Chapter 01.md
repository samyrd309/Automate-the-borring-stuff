## 5. This chapter introduced assignment statements, like spam = 10. What is the difference between an expression and a statement?

> An expression in Python evaluates to a single value, such as a number, string, or Boolean result, and can be used in various contexts like assignments or function arguments. In contrast, a statement performs an action or command but does not return a value itself
